# Asset licensing

Hetzly's code is MIT-licensed (see LICENSE). Two asset categories have their own terms, documented
here.

## Mascot sprites (Hetzi) — Elthen, redistribution permission granted ✅

The mascot sprite data in `Hetzly/Mascot/MascotFrameData+*.swift` is **converted/derived from
"2D Pixel Art Red Panda Sprites" by Elthen (Ahmet Avci)**,
https://elthen.itch.io/2d-pixel-art-red-panda-sprites.

The project owner made a purchase/donation to Elthen and **obtained approval to publish these
sprites as part of this open-source repository** (2026-07). This clears the previous
redistribution concern — the sprite data may be included in the public repo.

**Keep the attribution.** The "Mascot sprites by Elthen" credit in Settings → About must remain,
and this notice must stay in the repo, for as long as any Elthen-derived sprite ships in the app.

> Maintainers: keep the written approval (email / itch.io / Patreon message) on file so future
> contributors don't have to re-litigate this. A CC0-equivalent hand-drawn fallback also remains
> available under `scripts/mascot-fallback/` if the project ever wants to drop the dependency.

## App icon

The app icon (`app-icon.png` at the repository root, and the four
`Hetzly/Resources/Assets.xcassets/AppIcon*.appiconset/` catalogs generated from it) is an
**original logo provided by the project owner** and is covered by the repository's MIT license.
It is not derived from Elthen's sprites. (`AppIcon-Mono` is a grayscale rendering of the same
original logo.)

## Everything else

All other art (design system, generated share cards) is original to this project and covered by
the MIT license.
